Online Tutorial Website

Online tutoring web application that allows students to learn courses which are uploaded by teachers having video lectures and coding snippets verified by admins.
Functionalities for Students:
1) AI Search filter
2) Chatbot for instant query resolution and website guidance
3) Course Dynamic Pricing
4) Custom IDE supporting languages like Java,Python,C++
5) Try it now, to paste course code directly on IDE
6) Certification on Completion
7) Competitive Coding Playground to test students skills where students code will get tested on hidden test cases created by course teacher. 8) Recommendation of Courses by searches
9) Support of Paypal SDK for In-portal payment process.
Functionalities for Teachers:
1) Course creation by adding contents, videos and using IDE for codes snippets
2) Resolve Queries of Students for courses
3) Coding Playground Question and test cases creation
4) Teacher Performance dashboard
5) Tracking of profit earned from courses

Functionalities for Admins:
1) Verify course created by teachers before displaying to students
2) Track profit earned from website improvement charges applied on each course
3) Check All Teacher performance
